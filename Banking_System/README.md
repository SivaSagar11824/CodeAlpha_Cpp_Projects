# Banking System

## 📌 Description
This project simulates a simple banking system using Object-Oriented Programming in C++.

## 🚀 Features
- Create account
- Deposit money
- Withdraw money
- Display account details
- Menu-driven program

## 🛠️ Technologies Used
- C++
- OOP Concepts (Class & Object)

## ▶️ How to Run
1. Compile the program:
   g++ banking.cpp
2. Run:
   ./a.out

## 📷 Example
1. Enter account details
2. Perform deposit/withdraw operations
3. View updated balance
