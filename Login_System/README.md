# Login and Registration System

## 📌 Description
This project implements a basic login and registration system using file handling in C++.

## 🚀 Features
- User registration with username and password
- Stores credentials in a file
- Login verification system
- Displays success or error messages

## 🛠️ Technologies Used
- C++
- File Handling (fstream)

## ▶️ How to Run
1. Compile the program:
   g++ login.cpp
2. Run:
   ./a.out

## 📷 Example
1. Register a new user
2. Login using saved credentials

## 📁 File Used
- users.txt (stores username and password)
